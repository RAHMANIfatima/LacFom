Présentation du logiciel

Bienvenue dans le manuel d'utilisation de l'interface Web Lacfom, votre assistant numérique conçu pour simplifier et accélérer vos analyses. Ce guide complet vous fournira toutes les informations pour naviguer efficacement à travers les différentes fonctionnalités de l'application. Notre objectif est de rendre le processus de détection de contamination aussi intuitif et fluide que possible, en vous offrant un outil puissant et facile à maîtriser. Au fil de ce manuel, vous trouverez des descriptions textuelles précises des différents éléments de l'interface, tels que les boutons, les tableaux et les notifications. Nous avons également inclus des clarifications sur la signification des différents symboles que vous pourriez rencontrer, comme le checkmark indiquant une validation ou une conformité, et la croix signalant potentiellement une non-conformité ou une erreur. Nous sommes convaincus que ce manuel vous sera un allié précieux dans l'utilisation quotidienne de l'interface Web Lacfom, vous permettant ainsi de vous concentrer sur l'essentiel : l'analyse de vos résultats. N'hésitez pas à le consulter à tout moment pour répondre à vos questions ou pour vous remémorer une procédure spécifique.

Page d'Accueil

Lorsque vous lancez l'application, vous arrivez sur la page d'accueil. Vous y trouverez les éléments suivants :

  • Veuillez choisir un fichier : Une invitation à sélectionner les fichiers à analyser.
  • Choisir un fichier : Un bouton pour parcourir vos fichiers et sélectionner le fichier texte (.txt) que vous souhaitez analyser.
  • Importer : Un bouton pour importer le fichier sélectionné.
  • Paramètres : Un bouton pour accéder aux paramètres de l'application.
  • Manuel d'utilisation : Un bouton pour consulter ce manuel.

Modification des Paramètres

En cliquant sur le bouton Paramètres depuis la page d'accueil, vous accéderez à la page de modification des paramètres. Voici les options disponibles :

Paramètres de détection de contamination

  • Nombre minimum de marqueurs informatifs contaminés : La valeur par défaut est 2.
  • Hauteur des pics discriminant un allèle contaminé : La valeur par défaut est 1/3.

Paramètres éditeur

  • Émetteur : La valeur par défaut est PBP-P2A-GEN.
  • Entité d'application : La valeur par défaut est PBP-PTBM.

Paramètres kit

  • Changer de kit : Vous pouvez choisir un autre kit d'analyse en téléchargeant un fichier spécifique.
  • Choisir un fichier : Un bouton pour sélectionner un fichier de kit personnalisé.

Actions

  • Annuler : Annule les modifications.
  • Réinitialiser : Réinitialise tous les paramètres à leurs valeurs par défaut.
  • Valider : Enregistre les modifications et retourne à la page précédente.

Identification des Individus

Après avoir sélectionné un fichier, vous serez redirigé vers la page d'identification des individus. Voici ce que vous trouverez :

  • Succès : Une notification confirmant que le fichier a été importé avec succès.
  • Veuillez sélectionner les individus : Un rappel de l'action à effectuer.
  • Tableau d'association : n tableau listant les échantillons. Pour chaque échantillon, vous pouvez sélectionner le rôle correspondant (Mère, Fœtus, Père) en cliquant sur le cercle approprié.
  • Valider : Une fois l'association effectuée, vous pouvez valider pour passer à l'étape suivante.

Page des Résultats

Après avoir validé l'identification des individus, vous serez dirigé vers la page des résultats. Cette page affiche les conclusions de l'analyse.

  • Page des résultats : Le titre de la page.
  • Informations sur l'échantillon : Des informations détaillées sur les échantillons analysés, y compris les numéros d'échantillon pour le fœtus, la mère et le père.
  • Sexe fœtus : Le sexe présumé du fœtus (par exemple, M pour Masculin).
  • Témoin positif : Indique si le témoin positif est conforme (checkmark ou croix).
  • Témoin négatif : Indique si le témoin négatif est conforme (checkmark ou croix).
  • Concordance Mère/Fœtus : Indique s'il y a une concordance entre la mère et le fœtus (OUI/NON).
  • Concordance Père/Fœtus : Indique s'il y a une concordance entre le père et le fœtus (OUI/NON).
  • Tableau des résultats : Un tableau affichant les résultats pour chaque marqueur analysé. Les colonnes incluent :
    • Marqueur : Le nom du marqueur génétique.
    • Conclusion : Le résultat de l'analyse pour ce marqueur (par exemple, Contaminé, Non informatif).
    • Détails : Des informations supplémentaires sur le résultat.
  • Fichier : Un menu déroulant permettant d'exporter les résultats.
  • Options : Un menu déroulant offrant la possibilité d'accéder à la page Paramètres ainsi que de visualiser le fichier texte.

Consultez attentivement le tableau des résultats pour interpréter les conclusions de l'analyse.
